This is a Pipeline Report generated by the q pipeline management utility.

Left Top Pane

Select a master instructions file from the drop-down box. A master file is a top-level instructions file that controlled the execution of a series of pipeline jobs.

Left Middle Pane

Shows the instructions files and target scripts that defined the pipeline. These files are displayed in a tree that reflects the execution flow.

Clicking the name of a file will:
  a) display the contents of the q file or target script in the Right Pane.
  b) filter the Left Bottom Pane to show only those jobs executed by the selected file.

Left Bottom Pane

Shows status information for the jobs executed by the pipeline.

Clicking 'log', 'script' or 'env' will show the job's log file, execution script (as parsed by q) or environment variables, respectively.

Job dependency information is displayed such that the second job chain column contains the numbers of the predecessor jobs on which that job depended. An '@' symbol indicates an array job. Job wall time and RAM are reported for the single job or as the maximum usage by any job in an array.

Right Pane

Shows file contents, as well as these instructions.

Clicking hyperlinks in the Right Pane will show a target file or information on the version and usage of a system command.

Source Files

Source instructions files and target scripts are available in the 'instructions' folder present in the same folder that contained the html for this web page.
